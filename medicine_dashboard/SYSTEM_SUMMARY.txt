🔌 HARDWARE WIRING
==================

Arduino Pin Layout:

  Buzzer:
    Positive (Red) → Arduino Pin 9
    Negative (Black) → Arduino GND
  
  LED:
    Positive (Long leg) → Arduino Pin 13 (through 220Ω resistor)
    Negative (Short leg) → Arduino GND
  
  USB Connection:
    Arduino → PC via USB cable
    Auto-detects as COM port (usually COM3-COM5)


⚙️ SYSTEM WORKFLOW
==================

1. USER SETUP (One-time)
   • Upload Arduino code to board
   • Install Python packages
   • Start Flask server
   • Open web dashboard

2. MEDICINE SCHEDULE (One-time)
   • Enter medicine times (e.g., 09:00, 14:00, 22:10)
   • Times saved automatically
   • Can edit anytime

3. AUTOMATIC MODE (24/7)
   • Flask monitors time every second
   • When time matches medicine time:
     a. Sends "ALARM" to Arduino
     b. Arduino beeps buzzer 3 times
     c. LED blinks rapidly
     d. Web status shows "ACTIVE"
   • User clicks "OK" when medicine taken
   • System sends "STOP" command
   • Returns to idle, waits for next time

4. TESTING (Before relying on automation)
   • Use "TRIGGER ALARM NOW" button
   • Verifies buzzer, LED, web notifications work
   • Can test anytime without waiting

⏰ MEDICINE TIME FORMATS
========================

Always use 24-hour time format: HH:MM

Examples:
  ✓ 09:00  (9 AM morning dose)
  ✓ 12:30  (12:30 PM lunch dose)
  ✓ 14:00  (2 PM afternoon dose)
  ✓ 22:10  (10:10 PM evening dose)
  ✗ 9:00 (missing leading zero)
  ✗ 9 AM (12-hour format doesn't work)
  ✗ 900 (missing colon)

🔧 CUSTOMIZATION OPTIONS
========================

Modify Buzzer Pattern (Arduino code):
  • Change beep count: Line 48 "for (int i = 0; i < 3; i++)"
  • Change beep duration: Line 50 "delay(200)"
  • Make pattern more aggressive for alarms

Modify LED Blink Speed (Arduino code):
  • Change BLINK_INTERVAL: Line 12 "const int BLINK_INTERVAL = 300"
  • Lower = faster blink
  • Higher = slower blink

Modify Web Dashboard Colors:
  • Edit index.html style section
  • Change gradient colors, button colors, etc.
  • All CSS is customizable

Add More Medicine Times:
  • No limit to number of times
  • Just keep adding via web interface
  • Automatically saved

Change Server Port:
  • Flask default: 5000
  • Edit app.py line 155: port=5001
  • Then access: http://localhost:5001
